# Cluster 05: Two Pointers
**Tier 2: Logic** - Advanced two-pointer technique

## Pattern Overview
Use two pointers moving in same/opposite directions to solve array manipulation problems efficiently.

## CodeChef Practice Problems
- `Move_Zeros` - Move all zeros to end
- `Leaders` - Find all leaders in array
- `Snapchat` - Array transformation problem

## LeetCode Revision
- [LC 283](https://leetcode.com/problems/move-zeroes/) - Move Zeroes
- [LC 1299](https://leetcode.com/problems/replace-elements-with-greatest-element-on-right-side/) - Replace Elements with Greatest Element on Right Side

## Key Concepts
- Two-pointer approach
- In-place array modification
- Left-right scanning
- Right-to-left traversal
- Time Complexity: O(n), Space: O(1)
