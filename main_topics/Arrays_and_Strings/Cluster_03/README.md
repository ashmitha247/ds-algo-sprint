# Cluster 03: Neighbor Comparison
**Tier 1: Solo** - Comparing adjacent elements

## Pattern Overview
Problems that require comparing current element with neighbors to detect patterns like sorted arrays or duplicates.

## CodeChef Practice Problems
- `Check_Sorted` - Verify if array is sorted
- `Remove_Dups` - Remove duplicates from sorted array

## Key Concepts
- Adjacent element comparison
- Two-pointer technique (basic)
- In-place modification
- Time Complexity: O(n)
